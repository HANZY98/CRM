# Zoho Blueprint Pages

Upload this folder to GitHub.

Enable GitHub Pages:
Settings → Pages → Deploy from branch → main → /(root)

URLs become:

https://YOUR_GITHUB_USERNAME.github.io/REPO_NAME/verify-age/
https://YOUR_GITHUB_USERNAME.github.io/REPO_NAME/verify-residency/

Use those URLs in Zoho CRM widgets with External Hosting.
